Python is one of the most versatile languages, commonly used in machine learning and web development. In this class, students will gain both a concrete understanding of the foundations of Python and also the problem solving skills to use code to solve different problems.

This class is hands-on; students will learn concepts and then directly practice them by writing Python to solve algorithmic problems. To foster a nuanced understanding of the Python syntax, there will be friendly competition. For example, students who solve certain problems with the shortest code will earn recognition, which encourages experimentation of different techniques.

Artificial Intelligence (AI) is the buzzword of the century. What actually constitutes AI? In this course, we’ll survey the origins of AI all the way to today’s neural networks.

This class is a synthesis of the world-class TJHSST AI curriculum and our vast experience competiting in adversarial AI competitions such as MIT Battlecode and Correlation-One's Terminal.

Class Schedule

Note that there is a lot to AI. It is not possible to cover it all nor give the most robust understanding in a course of this scope. That being said, this course will give students the requisite ways to think about AI vs. traditional algorithms, construct these algorithms themselves. Most AI courses give a high-level understanding of the content, using libraries that solve the problem for you. We believe it’s important to start from the foundation.
